## 一、基本概念
### 1.Java初始化
	在 Java 语言中，当实例化对象时，对象所在类的所有成员变量首先要进行初始化，只有当所有类成员完成初始化后，
才会调用对象所在类的构造函数创建象。
#### 三个原则
	* 静态变量优先于非静态变量初始化，静态变量之初始化一次，非静态变量可能会初始化多次；
	- 父类优先于子类初始化；
	+ 按照成员变量的定义顺序初始化。即使变量定义散布于方法之中，也会被在被调用之前被初始化；
#### 数据类型转换（不考虑布尔类型 剩下：char，short，byte，int，long，float，double）
	基本数据类型：
	基本类型：byte 二进制位数：8   		 一个字节
	基本类型：short 二进制位数：16		两个字节
	基本类型：char 二进制位数：16  	两个字节
	基本类型：int 二进制位数：32		四个字节
	基本类型：float 二进制位数：32 		四个字节
	基本类型：long 二进制位数：64 		八个字节
	基本类型：double 二进制位数：64 	八个字节
	当char/short/byte之间进行运算时，默认值为int类型
	1、自动类型转换
	低精度向高精度转换，不会损失精度
	2、强制类型转换
	高精度向低精度转换，会损失精度，要使用强制类型转换符
#### 运算符
### 算数运算符
	正负号，+，-，*，/，%，++，--，连接符，=，+=，-=，*=，/=，%=
	++ -- 同js语法
### 逻辑运算符
	&，&&，|，||，！，^
	&和&&的区别：
	&无论左边是true还是false，右边都会进行运算。
	&&当左遍为fales，右边不再运行。
	一般使用&&。
	|与||的区别：
	| 无论左边是true还是false，右边都会进行运算。
	|| 左边为true时，右边不再运行。
### 位运算符
